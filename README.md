
## neteaseAnswerSystem

前后端分离式仿 NetEase 答题系统

### 前端
Vue.js + uniapp

### 后端
Spring boot + Mysql + Mybatis + Druid + Jwt
<br><br>
Spring boot 目录 ：`__springboot`

Mysql 目录 ：`__mysql`
